# Exercise 6.3-6.19

A simple anecdotes frontend from a [template](https://github.com/fullstack-hy2020/redux-anecdotes) that provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## How to Run the App

To run this app, follow these steps:

Install the dependencies:

```bash
npm install
```

Start the frontend development server:

```bash
npm run dev
```
