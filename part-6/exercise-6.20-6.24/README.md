# Exercise 6.20-6.24

A simple anecdotes frontend from a [template](https://github.com/fullstack-hy2020/query-anecdotes) that provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## How to Run the App

To run this app, follow these steps:

Install the dependencies:

```bash
npm install
```

Start the backend server that uses Json Server:

```bash
npm run server
```

Start the frontend development server:

```bash
npm run dev
```
