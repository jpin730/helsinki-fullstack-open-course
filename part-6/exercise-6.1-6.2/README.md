# Exercise 6.1-6.2

A simple counter frontend from a [template](https://github.com/fullstack-hy2020/unicafe-redux) that provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## How to Run the App

To run this app, follow these steps:

Install the dependencies:

```bash
npm install
```

Start the frontend development server:

```bash
npm run dev
```

<!-- tests -->

To run the tests, use the following command:

```bash
npm test
```
