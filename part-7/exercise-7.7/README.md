# Exercise 7.7

A simple country frontend from a [template](https://github.com/fullstack-hy2020/country-hook) that provides a minimal setup to get React working in Vite with HMR.

## How to Run the App

To run this app, follow these steps:

Install the dependencies:

```bash
npm install
```

Start the frontend development server:

```bash
npm run dev
```
